## 使用 .toggle() 切換狀態
> 在 jQuery 1.9以上版本，toggle() ，無法用 [callback function](https://expect7.pixnet.net/blog/post/38296922)，僅能做到顯示與隱藏。如果想要進行特定的樣式切換，可以透過 toggleClass() 增加或移除 class 以達到效果。  
對應檔案：`Q2-jQuery-toggle.html`
